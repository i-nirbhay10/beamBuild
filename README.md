# BeamBuild

BeamBuild is a comprehensive React Native application designed for construction management. It helps teams track projects, manage tasks, coordinate with team members, and handle documents effectively.

## Features

- **Project Management**: Track budgets, timelines, and overall progress.
- **Task Management**: Assign and track tasks for various team members.
- **Team Coordination**: Manage different teams and permissions.
- **Document Storage**: Upload and manage blueprints, contracts, and permits.
- **Messaging & Notifications**: Stay updated with real-time messages and alerts.

## Getting Started

### Prerequisites

- Node.js
- React Native environment (Android Studio / Xcode)

### Installation

1. Clone the repository.
2. Run `npm install` or `yarn install` to install dependencies.
3. Start the Metro server with `npm start`.
4. Run the app on Android with `npm run android` or on iOS with `npm run ios`.
